# 🏦 Bank Management System (C++)

A simple console-based Bank Management System developed in C++. This project demonstrates the fundamentals of Object-Oriented Programming (OOP) and menu-driven programming.

## 📌 Features

- Create a bank account
- Deposit money
- Withdraw money
- Display account details
- Input validation for invalid menu choices
- Simple and easy-to-understand code

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
- VS Code / Any C++ Compiler

## 📂 Project Structure

```
BankManagement.cpp
README.md
```

## ▶️ How to Run

### Compile

```bash
g++ BankManagement.cpp -o bank
```

### Run

```bash
./bank
```
